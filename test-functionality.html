JSON Formatter - Comprehensive Functionality Test

Test Cases

1. Valid JSON - Simple Object

{"name":"John","age":30,"city":"New York"}

Expected: Should format with proper indentation

2. Valid JSON - Nested Object

{"user":{"name":"Alice","contacts":{"email":"alice@example.com","phone":"+1234567890"},"preferences":{"theme":"dark","notifications":true}},"timestamp":1234567890}

Expected: Should format with nested indentation

3. Valid JSON - Array

[1,2,3,{"id":4,"value":"four"},["nested","array",{"deep":true}]]

Expected: Should format arrays with proper brackets

4. Invalid JSON - Missing Quote

{name:"John","age":30}

Expected: Should show error message

5. Invalid JSON - Trailing Comma

{"name":"John","age":30,}

Expected: Should show error message

6. Edge Case - Empty Object

{}

Expected: Should format as empty object

7. Edge Case - Unicode and Special Characters

{"emoji":"😀","unicode":"你好","special":"tab\there\nnewline","quote":"She said \"Hello\""}

Expected: Should handle special characters correctly

8. Large JSON

{"data":[{"id":1,"name":"Item 1"},{"id":2,"name":"Item 2"},{"id":3,"name":"Item 3"},{"id":4,"name":"Item 4"},{"id":5,"name":"Item 5"},{"id":6,"name":"Item 6"},{"id":7,"name":"Item 7"},{"id":8,"name":"Item 8"},{"id":9,"name":"Item 9"},{"id":10,"name":"Item 10"}],"metadata":{"total":10,"page":1,"per_page":10}}

Expected: Should handle large JSON efficiently
