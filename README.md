## ✨ Features

### 🛠️ Core JSON Tools
- **Smart JSON Formatter** - Beautify JSON with customizable indentation
- **JSON Minifier** - Compress JSON to single line
- **Real-time Validator** - Instant error detection with line numbers
- **JSON to CSV Converter** - Transform JSON arrays to CSV format
- **JSON Diff Tool** - Compare two JSON files side by side
- **Schema Generator** - Auto-generate JSON Schema
- **Path Finder** - Click any property to get its JSON path
- **Escape/Unescape** - Handle JSON string escaping
- **Sort Keys** - Alphabetically sort object keys
- **Remove Nulls** - Clean up null values

## 🚀 Quick Start

### Option 1: Using Python (No dependencies)
```bash
python3 -m http.server 8080
```
Then open http://localhost:8080

### Option 2: Using Node.js
```bash
npm install
npm start
```
Then open http://localhost:8080

### Option 3: Using npx (No installation)
```bash
npx serve .
```

### Option 4: Direct File Access
Simply open `index.html` in your web browser
